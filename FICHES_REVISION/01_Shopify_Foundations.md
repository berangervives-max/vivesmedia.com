## Format de l'examen
- QCM en ligne, ~40-60 questions, score requis ~70 %.
- Repassable si échec. Pas de piège technique : ce sont les **fondamentaux du commerce + plateforme Shopify**.

## 1. L'écosystème Shopify
- **Shopify** = plateforme SaaS de commerce (hébergée, abonnement mensuel).
- Offres : **Basic / Shopify / Advanced / Plus** (+ Starter). Plus = grandes marques, checkout personnalisable, Functions.
- **Admin Shopify** = back-office (produits, commandes, clients, analytics, apps, thèmes).
- **Online Store** = canal vitrine ; autres canaux : POS, réseaux sociaux, marketplaces (multichannel).

## 2. Produits & collections
- **Produit** : titre, description, images, prix, **variantes** (taille/couleur), **SKU**, suivi de stock.
- **Collections** : **manuelles** (sélection à la main) vs **automatisées** (règles : tag, prix, type…).
- **Metafields** : champs personnalisés pour enrichir produits/collections (clé pour le sur-mesure).

## 3. Online Store & thèmes
- **Thèmes** : Liquid + sections/blocks ; éditeur **« Sections everywhere »** (Online Store 2.0).
- Pages, articles de blog, menus de navigation, **domaine** personnalisé.
- Vitesse + mobile = priorité (lien direct avec ton argumentaire de prospection !).

## 4. Commandes & clients
- Cycle commande : **paiement → traitement (fulfillment) → expédition → livraison**.
- Statuts : payé/non payé, traité/non traité, remboursé.
- Clients : fiches, segments, comptes clients ; **Shopify Payments**, passerelles tierces, **taxes & expédition**.

## 5. Apps & App Store
- Extensions de fonctionnalités via le **Shopify App Store**.
- Notion d'**embedded apps**, APIs (Admin API, Storefront API → Hydrogen).

## 6. Partenaire & métier
- **Partner Dashboard** : test stores, transferts de boutiques, commissions/revenus partenaires.
- Bonnes pratiques : sécurité, RGPD/CGV, accessibilité, performance.

## Points qui tombent souvent (pièges QCM)
- Différence **collection manuelle vs automatisée**.
- Ce que permet **Plus** (vs Basic/Advanced) : checkout.liquid/Functions, multi-boutiques.
- **Variantes** : max 100 par produit (3 options) — limite classique.
- **Storefront API** (headless/Hydrogen) ≠ **Admin API** (gestion back-office).
- Canaux de vente multiples = **multichannel** (pas seulement le site).
